# Chance-of-Admission-Prediction
Machine Learning project to predict chance of admission using Linear Regression

This project predicts the probability of admission of a student into a graduate program using Machine Learning (Linear Regression).

#Features Used
- GRE Score
- TOEFL Score
- University Rating
- SOP
- LOR
- CGPA
- Research

#Model Used
Linear Regression

#Evaluation Metrics
- MAE
- MSE
- MAPE

#Conclusion
The model successfully predicts admission probability with low error values and demonstrates good performance.
